This is a program for Tic Tac Toe game in Java
To run this program,
run "java TicTacToe.java" command
Program will first ask to choose option 
1. Player vs Player
2. Player vs computer

You can choose any of the above options

if option choosen is 1, then player 1 is 'X' and player 2 is 'O' and player 1 goes first
if option choosen is 2, then player 1 is 'X and computer is 'O' and player goes first

Enjoy my game :)
